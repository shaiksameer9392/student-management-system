# 🎓 Student Management System

A responsive Student Management System built using **Angular, TypeScript, HTML, CSS, and Bootstrap**. The application allows users to manage student records with CRUD operations, search functionality, form validation, and local storage.

---

## 🚀 Features

- ➕ Add Student
- ✏️ Update Student
- 🗑️ Delete Student
- 🔍 Search Student by ID
- ✅ Form Validation
- 💾 Local Storage
- 📱 Responsive Dashboard UI
- 🎨 Modern User Interface

---

## 🛠️ Technologies Used

- Angular
- TypeScript
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Local Storage

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/shaiksameer9392/student-management-system.git
```

Go to the project folder:

```bash
cd student-management-system
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
ng serve
```

Open your browser:

```
http://localhost:4200
```
